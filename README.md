# Projeto Ágil - Sistema de Tarefas

## Descrição
API para gerenciamento de tarefas usando Flask e MySQL.

## Funcionalidades
- Criar tarefas
- Listar tarefas
- Atualizar tarefas
- Deletar tarefas

## Tecnologias
- Python
- Flask
- MySQL

## Metodologia
Kanban (To Do, In Progress, Done)

## Como rodar
python app.py
